# Grocery-Store-Management-System

A simple Python-based Grocery Store Management System that manages product inventory and billing.

## Features

- Add Products
- View Products
- Update Stock
- Delete Products
- Generate Bills
- Inventory Tracking

## Technologies Used

- Python
- JSON (for data storage)

## Project Structure

main.py - Main program  
inventory.py - Product management  
billing.py - Billing system  
database.py - Data handling  
data/products.json - Product storage

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python main.py
